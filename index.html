The Ultimate Node.js API Framework

Welcome to Crudios

crudios is a powerful Node.js API framework designed to help you build scalable, maintainable, and high-performance APIs with ease. It includes a no-code dashboard for creating services, a database-agnostic interface, and a rich set of built-in tools.

Key Features

No-Code Dashboard

Create and manage services interactively without CLI.

Database Agnostic

Supports MongoDB, Cassandra, MySQL, PostgreSQL, SQLite, MariaDB with a unified interface.

Fastify Under the Hood

Leverage Fastify's speed and performance for your APIs.

VineJS Schema Validation

Single schema works across all databases for validation.

Built-In Tools

Includes caching, rate limiting, encryption, file uploads, and more.

Authorization Management

RBAC or ACL-based authorization out of the box.

Advanced Features

  • ✓ Database Visualizer and Editor
  • ✓ Config Management and Type-Safe Environment Variables
  • ✓ Thoughtful Folder Structure for Maintainability
  • ✓ Routing, Controllers, and Middleware Support
  • ✓ File Uploads with FlyDrive
  • ✓ Type-Safe Event Emitter
  • ✓ Built-In Server Status Check API
  • ✓ Built-In Mail Support with NodeMailer
